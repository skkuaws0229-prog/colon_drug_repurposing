코멘트 — v1 vs v3 불일치

v1의 3개 약물 모두 v3 Top 15에 포함되지 않습니다. 이건 방법론 차이 때문입니다. v1은 target gene expression 기반 검증이었고, v3은 IC50 예측 모델 + 6단계 확장 평가를 거쳤습니다. v3가 더 정교한 모델을 사용했으므로 결과가 달라지는 것은 자연스럽습니다.

이건 v1이 틀렸다는 뜻이 아닙니다. 평가 관점이 다릅니다. v1의 약물들이 나쁜 것이 아니라, v3의 평가 체계에서 더 높은 순위의 약물이 발견된 것입니다.
